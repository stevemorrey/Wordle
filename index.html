How to set the word of the day

To specify the word for a particular date, edit daily-words.json and add an entry like: 

{
  "2026-01-06": "BRAVE",
  "2026-01-07": "CLEAR"
}

If no entry exists for today, the app will pick a word deterministically from words.json based on the date.
